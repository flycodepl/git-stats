git-state
======
Show basic statistic for your project

INSTALL
======

```sh
$ curl -L https://github.com/flycodepl/git-stats/raw/master/install | sh
```

Usage
======

```sh
$ cd path/to/your/project
$ git state

Paul Peter Flis:
	lines: 57
	participation: 100.0000%

Summary:
	line: 57
	files: 2

```
